SQL, Structure Query(Lenguege de Consulta Estructurado) es un lenguaje de programacion para trabajar con base de datos relacionales como MySQL, Oracle, etc.
MySQL es un interpetador de SQL, es un servidor de base datos.
MySQL permite crear base de datos y tablas, insertar datos, modificaciones, eliminarlos, ordenalos, hacer consultas y realizar muchas operaciones, etc., resumiendo: administrar base de datos.

Este tutorial por objetivo acercar los conceptos iniciales para introducirse en el mundo de las bases de datos
